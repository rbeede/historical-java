Project #5:    Zip up and submit the following files: CalculateGrade.java, CS1323Grades.java, and EasyIn.java (provided for you)
 
Due Date:    Midnight, Friday, March 10, 2000
 
Assignment: Using the EasyIn class for input and output (not the javabook classes), write a program called CalculateGrade that will prompt the user to input all of their grades in CS 1323, calculate and display averages for each subcategory (i.e. quizzes, exams, projects….) , and then calculate and display the Course Average and Course Letter Grade.
  
General information needed for calculations:
GRADE SUBCATEGORIES PERCENTAGES   Grade Scale
Projects 20%   90-100 = A
Quizzes 10%   80-90 = B
Lab Attendance 10%   70-80 = C
Lab Final 10%   60-70 = D
Exam I and II 30%   0-60 = F
Final Exam 20%    
TOTAL POSSIBLE 100%    

 

Steps for the main class called CalculateGrade (use an EasyIn object for input and output):
  1. Declare a CS1323Grades object
  2. Instantiate the CS1323Grades object, passing the constructor all of the percentages of each subcategory of grades
  3. Prompt the user for the number of projects they have taken thus far
  4. Call a public method from the CS1323Grades class called calculateProjectAverage( ) and pass the number of projects as an argument
  5. Prompt the user for the number of quizzes they have taken thus far
  6. Call a public method from the CS1323Grades class called calculateQuizAverage( ) and pass the number of quizzes as an argument
  7. Prompt the user for the number of labs they have missed thus far
  8. Call a public method from the CS1323Grades class called calculateAttendanceGrade( )
  9. Prompt the user to enter a 'y' or 'n' based on whether they have taken the lab final yet
    • If they answered yes, call a public method from the CS1323Grades class called calculateLabFinalGrade( )
  10. Prompt the user for the number of exams (out of two) they have taken this far
  11. Call a public method from the CS1323Grades class called calculateExamAverage( ) and pass the number of exams as an argument
  12. Prompt the user to enter a 'y' or 'n' based on whether they have taken the Final Exam yet
    • If they answered yes, call a public method from the CS1323Grades called calculateFinalExamGrade( )
  13. Finally, call a public method from the CS1323Grades class called calculateCourseGrade( ) that returns the course average
  14. Calculate the course letter grade and then display the course average and course letter grade
